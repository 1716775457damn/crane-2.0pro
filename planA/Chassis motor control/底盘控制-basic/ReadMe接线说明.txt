接线说明：(电机驱动板需要5V供电，下面的接线例子，我们从主控板上取5V电源给电机驱动板供电)

Arduino	-	电机驱动板
TX0	-	RX
RX0	-	TX
VCC	-	VCC
GND	-	GND


注意事项：
1. 既然例子是从主控板取5V电源，你的主控板首先要供电5V
2. 电机驱动板需要两个电源，一个是24V给电机驱动用，一个5V是给板载处理器供电用，驱动板上不带24V降压5V电路！
3. 例子用到了串口0，众所周知，串口0是Arduino 2560下载程序的口，所以你主控版下载程序的时候，请把主控板上所有的线路拔掉，防止占用串口0,
